# Stream Distribution Portal. BE

## Project setup
```
npm install
docker-compose up && docker-compose exec application composer install
```

## Project start

    npm run dev
    docker-compose up

  or

    npm run dev
    docker-compose up -d

## Docker tools

- PhpMyAdmin - http://localhost:3001
- Mailhog - http://localhost:3002
- Redis Admin UI - http://localhost:3003
